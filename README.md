# Music Genre Clustering using VAE

## Project Overview
This project implements Variational Autoencoder (VAE) for music genre clustering using the GTZAN dataset. The goal is to extract latent features from audio and perform clustering to group similar music genres.

## Dataset
GTZAN Dataset: 1000 audio tracks, 10 genres (blues, classical, country, disco, hiphop, jazz, metal, pop, reggae, rock)
